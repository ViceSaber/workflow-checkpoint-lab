# UI Redesign: Dark Theme Kanban Board

## Summary

Redesign the task kanban board from a light-themed simple layout to a modern dark-themed professional UI with sidebar navigation, enriched task cards, and polished visual details.

## Motivation

The current light-themed UI is functional but visually basic. A modern dark theme with professional design elements (sidebar, colored tags, avatars, progress bars, due dates) will significantly improve the user experience and visual appeal.

## Scope

- Full dark theme CSS rewrite
- New Sidebar component (decorative navigation)
- Enriched TaskCard with tags, avatars, progress bars, due dates
- Extended data model (dueDate, assignee, tags)
- Updated TaskFormModal with new fields
- Dark theme for all existing pages (detail, editor, comments)
